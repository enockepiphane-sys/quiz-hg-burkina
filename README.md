# Quiz HG Burkina 🇧🇫

Une application interactive de quiz en Histoire-Géographie spécialement conçue pour les élèves des classes de Terminale au Burkina Faso. Ce projet vise à offrir un outil de révision moderne, accessible et efficace pour préparer l'examen du Baccalauréat.

---

## 🚀 Fonctionnalités

* *Quiz Thématiques* : Questions basées rigoureusement sur le programme officiel d'Histoire et de Géographie des classes de Terminale.
* *Correction Instantanée* : Explications des réponses pour favoriser un apprentissage autonome.
* *Suivi des Scores* : Permet aux élèves d'évaluer leur niveau et de progresser à leur rythme.
* *Interface Fluide* : Un design épuré, rapide et adapté aux téléphones comme aux ordinateurs.

## 📁 Structure du Projet

* README.md : Présentation du projet.
* questionnaire_hg_terminale-2.md (ou .json) : Base de données des questions et réponses d'Histoire-Géo.

## 🛠️ Technologies Utilisées

(Exemple à adapter selon ton code)
* *Frontend* : HTML5, CSS3, JavaScript (ES6)
* *Hébergement* : GitHub Pages (optionnel)

## 🎯 Objectifs du Projet

L'éducation est la clé du développement. Ce quiz a pour but de démocratiser l'accès aux ressources de révision de qualité pour tous les lycéens burkinabè, en transformant l'apprentissage en une expérience interactive et stimulante.
